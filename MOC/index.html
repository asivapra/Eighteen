What's This?

This is a web interface to create a mail tracking code that will enable you to get an email notification when your outbound mail has been read by the recipient.

How to do:

  • Login to your account, if not already done.
  • Type in or choose from the dropdown list the mail recipient address.
  • Click 'Copy to Clipboard' the image link.
  • Insert the image link anywhere in your outbound mail.
